# Aether Knight Frontend

React + TypeScript client for the [Aether Knight](https://github.com/) voice AI assistant.

## Quick start

```bash
cd frontend
cp .env.example .env
npm install
npm run dev
```

See [frontend/README.md](frontend/README.md) for full documentation.

**Backend** must be running at `http://127.0.0.1:8000` (configure via `VITE_API_URL`).
